Home

HTTP Methods Path Description
GET /(Home page) Takes you to the home page

Users Table

HTTP Methods Path Description
GET /users displays a list of users
GET /user/:id displays a specific user based on id
POST /register Creates a new user
POST /login Logs in a single user with their details
PATCH/PUT /user/:id Updates an existing user or adds a user
DELETE /user/:id Deletes a user

Products Table

HTTP Methods Path Description
GET /products Displays a list of products
GET /products/:id Displays a single user by id
POST /product Adds a new product
PUT/PATCH /products/:id Updates a products details
DELETE /products/:id deletes a product from the database

Cart Table

HTTP Methods Path Description
GET /cart Displays a list of all orders made
GET /user/:id/carts Displays orders made by a single user
POST /user/:id/cart Adds a product to the cart
PATCH /user/:id/cart/:id Updates a product in the cart
DELETE /user/:id/cart Clears a users cart
DELETE /user/:id/cart/:id Deletes a single item from the users cart
